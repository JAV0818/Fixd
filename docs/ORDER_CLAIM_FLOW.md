# Order Claim Flow - "Claim to Chat" System

## Overview
This system ensures only ONE provider can contact a customer about a specific order at any time, while still allowing providers to ask clarifying questions before committing.

---

## Status Definitions

| Status | Description | Who Can See | Who Can Chat |
|--------|-------------|-------------|--------------|
| `Pending` | New order, no one assigned | All providers | No one |
| `Claimed` | Provider is reviewing | All providers (grayed out) | Only claiming provider |
| `Accepted` | Provider committed to job | Only assigned provider | Only assigned provider |
| `InProgress` | Service being performed | Only assigned provider | Only assigned provider |
| `Completed` | Service finished | Historical | Chat closed |
| `Cancelled` | Order cancelled | Historical | Chat closed |
